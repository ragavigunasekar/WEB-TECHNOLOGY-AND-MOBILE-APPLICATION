# Email Sender Android App

## AIM

To develop a mobile application that allows the user to send an email using the device's installed email client.

## DESCRIPTION

This Android application provides a simple interface where users can enter the recipient's email address, subject, and message content. When the user clicks the **Send Email** button, the application opens the default email client with the entered information pre-filled, allowing the user to send the email easily.

## FEATURES

* Enter recipient email address
* Enter subject and message
* Send email using device email client
* Simple and user-friendly interface

## TECHNOLOGIES USED

* Android Studio
* Java
* XML
* Android Intent API

## HOW TO RUN

1. Open the project in Android Studio.
2. Build and run the application on an emulator or Android device.
3. Enter the recipient email, subject, and message.
4. Click **Send Email**.
5. The email client will open with the entered details.

## RESULT

The mobile application successfully sends an email by opening the installed email client with the provided information.
