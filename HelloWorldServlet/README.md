# Hello World Servlet

## Aim
To create a basic Java Servlet that prints "Hello, World!" in a web browser.

## Technologies Used
- Java
- Servlet API
- Apache Tomcat

## How to Run

1. Compile the servlet:
   javac HelloServlet.java

2. Place the .class file inside:
   WEB-INF/classes/

3. Start Tomcat server.

4. Open browser and visit:
   http://localhost:8080/YourProjectName/hello

## Output
The browser displays:

Hello, World!

## Result
The Servlet program executed successfully and displayed the message in the browser.
