# Web Page Creation using HTML

## AIM
To create a web page that embeds a map with hotspots, frames, and links using HTML.
The page uses inline, embedded, and external CSS for styling.

## TECHNOLOGIES USED
- HTML5
- CSS3

## FEATURES
- Chennai city map with clickable hotspots
- Embedded frame using iframe
- Navigation links
- Inline, Embedded, and External CSS

## ALGORITHM
1. Create the basic HTML structure.
2. Link external CSS and add embedded styles.
3. Display a Chennai map using `<img>`.
4. Create hotspots using `<map>` and `<area>`.
5. Embed external content using `<iframe>`.
6. Add navigation and external links.
7. Apply CSS styles using inline, embedded, and external methods.
8. Test the webpage.

## OUTPUT
A webpage displaying a Chennai map with hotspots, frames, and links.

## RESULT
HTML page is created successfully with embedded Chennai map along with hotspots,
frames, and links.
